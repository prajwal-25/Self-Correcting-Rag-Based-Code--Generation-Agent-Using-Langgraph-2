Key Features

RAG-Powered Context

Retrieves relevant Python documentation from a FAISS vector store using HuggingFace embeddings, providing rich context for accurate code generation.

FAISS · HuggingFace · WebBaseLoader

Self-Correcting Loop

Generated code is executed and tested. If errors are found, the agent automatically retries with error context — up to 3 iterations.

StateGraph · Conditional Edges

Structured Output

Every response follows a clean schema: problem description, import statements, and functional code — ready to copy and run.

Pydantic · CodeSchema

Built on LangGraph's StateGraph with checkpointing and memory, enabling complex multi-step agent workflows with state persistence.

LangGraph · InMemorySaver

ChatMistralAI · mistral-large-latest
